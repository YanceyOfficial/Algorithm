## 题目

给你一个只包含 `(` 和 `)` 的字符串, 找出最长有效(格式正确且连续)括号子串的长度.

:::info 示例
输入: `s = )()())`

输出: `4`

解释: 最长有效括号子串是 `()()`
:::

```ts
/**
 * @param {string} s
 * @return {number}
 */
var longestValidParentheses = function (s) {
  const n = s.length
  const stack = [-1]

  let max = 0
  for (let i = 0; i < n; i++) {
    const curr = s[i]
    if (curr === '(') {
      stack.push(i)
    } else {
      stack.pop()

      if (stack.length !== 0) {
        max = Math.max(i - stack[stack.length - 1], max)
      } else {
        stack.push(i)
      }
    }
  }

  return max
}
```

```rust
use std::cmp;

pub fn longest_valid_parentheses(s: String) -> i32 {
    let mut max = 0;
    let mut stack = vec![-1];

    for (i, ch) in s.chars().enumerate() {
        if ch == '(' {
            stack.push(i as i32);
        } else {
            stack.pop();

            if stack.len() != 0 {
                max = cmp::max(max, i as i32 - stack.last().unwrap());
            } else {
                stack.push(i as i32);
            }
        }
    }

    max
}
```
