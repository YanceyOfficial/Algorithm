## 题目

给定一个包含**非负数的数组**和一个**目标整数 k**, 编写一个函数来判断该数组是否含**有连续的子数组**, 其大小至少为 2, 且总和为 k 的倍数, 即总和为 `n * k`, 其中 n 也是一个整数.

:::info 示例

```ts
输入: [23, 2, 4, 6, 7], k = 6
输出: true
解释: [2, 4] 是长度为 2 的子数组, 并且和为 6, 是 k 的倍数, 符合要求.
```

```ts
输入: [23, 2, 6, 4, 7], k = 6
输出: true
解释: [23, 2, 6, 4, 7] 是长度为 5 的子数组, 并且和为 42, 是 k 的倍数, 符合要求.
```

:::
