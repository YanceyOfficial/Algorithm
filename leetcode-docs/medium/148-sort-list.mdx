## 题目

给你链表的头结点 head, 请将其按**升序**排列并返回**排序后的链表**. 要求在在 O(nlogn) 时间复杂度和常数级空间复杂度下, 对链表进行排序.

提示：

- 链表中节点的数目在范围 [0, 5 * 10<sup>4</sup>] 内
- -10<sup>5</sup> <= Node.val <= 10<sup>5</sup>

:::info 示例

输入: -1 -> 5 -> 3 -> 4 -> 0

输出: -1 -> 0 -> 3 -> 4 -> 5
:::
